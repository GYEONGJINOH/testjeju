가족에게 마음을 전하는 안부 한 통

청년 · 직장인 세대를 위한 가족 안부 전화 웹 서비스

1기획 배경

우리는 왜 가족에게 전화를 못 드릴까?

😰 지금의 현실

계속되는 경쟁, 이어가야 할 커리어와 업무. 학업·직장·사업으로 고향이 아닌 타지에 사는 청년이 늘고 있습니다. 나를 제외한 주위에 대한 관심이 줄고, 그만큼 가족과의 대화와 만남도 줄어듭니다.

➜
🌳 문제의 본질

마음이 없어서가 아닙니다. 잊고, 미루고, 계기가 없어서입니다. 그렇다면 때맞춰 상기시키고, 마음을 데우고, 그 자리에서 바로 행동하게 만들면 됩니다.

2핵심 아이디어 — 3초 안에 전화까지

알림에서 통화까지의 거리를 최대한 짧게 만드는 것이 이 서비스의 전부입니다.

때⏰ 알림이 울린다

내가 정한 주기와 시간대에
맞춰 알려줍니다

마음🖼️ 추억 사진이 뜬다

부모님·조부모님과 찍은
사진이 원목 액자에

행동📞 밀어서 전화

슬라이드 한 번으로
바로 통화 연결

💡 왜 "밀어서" 인가? 사진을 넘기다 실수로 전화가 걸리면 오히려 부담이 됩니다. 그래서 넘기기(사진 탐색)와 밀기(전화 연결)를 분리해, 의도한 순간에만 연결되도록 설계했습니다.

3주요 기능

기획한 기능을 모두 구현했습니다.

⏰맞춤 알림

전화 주기(주 N회)와 시간대를 직접 설정. 앱을 꺼둬도 이메일로 알림이 갑니다.

🖼️추억 사진 액자

가족 사진을 원목 액자로 슬라이드. 기기에서 바로 업로드(구글 드라이브 저장).

📞밀어서 전화 + 자동 기록

저장된 전화번호로 즉시 연결하고, 통화 시도를 자동으로 카운팅합니다.

📊2D 데이터 시각화

일별 막대그래프, 가족별 원형그래프, 주간 목표 진행바, 🔥 연속 달성 주(스트릭).

📍가족 위치 공유

직선거리 계산과 길찾기 연결. ON/OFF 토글로 언제든 중단 가능.

🎧고객 지원

고객센터(인터넷전화·이메일·팩스)와 FAQ, 1:1 문의와 답변 조회.

이용을 뒷받침하는 기능

  • 회원가입 / 로그인 — 사용자별로 연락처·통계·설정이 완전히 분리
  • 비밀번호 재설정 — 이메일로 링크 발송 (1시간 만료 · 1회용)
  • 이용약관 · 개인정보처리방침 페이지

4전화 빈도, 숫자로 보여주기

"얼마나 자주 연락드렸나"를 감이 아니라 데이터로 확인합니다.

4전체 / 이번 주 / 이번 달 / 목표 달성률
30일일별 전화 횟수 막대그래프
👨‍👩‍👧가족별 전화 비중 원형그래프
🔥주간 목표 연속 달성 기록
🎯 단순 기록을 넘어 행동을 유도합니다. 남은 날짜보다 남은 횟수가 많으면 "남은 3일 동안 2번 더! 서둘러 전화드려요" 처럼 경고가 뜨고, 이메일 알림의 문구도 상황에 따라 강도가 달라집니다.

구성하는 일비고

🌳 이름과 로고안부나무 — 안부가 쌓여 자라는 나무. 로고는 나무 안의 하트

7개인정보 보호 설계

위치와 연락처를 다루는 서비스인 만큼, 설계 단계부터 반영했습니다.

항목설계
비밀번호SHA-256 + 솔트 해시로 저장. 평문은 어디에도 남지 않습니다.
위치 공유사용자가 직접 ON 한 경우에만 수집·공유. 언제든 OFF로 즉시 중단.
공유 범위가족 연동 코드가 같은 사람끼리만. 모르는 사람에게 노출되지 않습니다.
거리 계산내 좌표는 내 기기에만 저장. 공유를 꺼도 가족과의 거리는 볼 수 있습니다.
세션 · 토큰로그인 세션 30일 만료. 재설정 토큰은 1시간 · 1회용이며 사용 시 기존 세션 전부 무효화.

8개발 현황

실제 배포본으로 전 과정을 테스트해 확인했습니다.

구현 완료 · 동작 확인

  • 회원가입 → 로그인 → 새로고침 후 세션 유지
  • 가족 연락처 등록 → 사진 액자 표시
  • 전화 기록 → 통계 반영 (목표 달성률 · 진행바 · 스트릭)
  • 일별 · 가족별 그래프 렌더링
  • 1:1 문의 접수 → 내역 조회
  • 화면과 서버 간 21개 요청 전부 정상 연결

남은 과제

  • 화면 호스팅 — 현재 HTML이 PC 안의 파일이라, 휴대폰에서 열어야 동작하는 전화 연결(tel:)과 가족의 접속이 아직 불가능합니다.
  • 비밀번호 재설정 링크 — 화면 주소가 정해지면 완성됩니다.
⚠️ 솔직한 현재 상태 — 기능은 모두 구현되어 동작하지만, 이 서비스의 핵심인 "실제로 전화가 걸리는" 경험은 호스팅 이후에 완성됩니다. 세 가지 남은 문제의 원인이 모두 "화면이 아직 인터넷에 올라가 있지 않다" 하나로 모입니다.
